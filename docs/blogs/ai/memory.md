# 记忆系统与RAG
记忆系统缺失引发的问题：<br/>
1、上下文缺失：长对话，早期重要信息因上下文窗口限制丢失。<br/>
2、个性化缺失：Agent无法记住用户的偏好、习惯或特定需求。<br/>
3、学习能力受限：无法从过往失败或成功经验中学习改进。<br/>
4、一致性问题：多轮对话，出现前后矛盾的回答。<br/>
RAG系统缺失的问题：<br/>
1、知识时效性：大模型的训练数据有时间截止点，无法获取最新信息<br/>
2.专业领域知识深度不足<br/>
核心思想:在模型生成回答之前，先从一个外部知识库（如文档、数据库、API）中检索出最相关的信息，并将这些信息作为上下文一同提供给模型。

## 记忆系统

### 工作记忆
工作记忆是记忆系统中最活跃的部分，它负责存储当前对话会话中的临时信息。工作记忆的设计重点在于快速访问和自动清理。<br>

### 情景记忆
情景记忆负责存储具体的事件和经历，它的设计重点在于保持事件的完整性和时间序列关系。
### 语义记忆
语义记忆是记忆系统中最复杂的部分，它负责存储抽象的概念、规则和知识.
### 感知记忆
感知记忆支持文本、图像、音频等多种模态的数据存储和检索。
