## 什么是Agent?
任何能通过**传感器**感知其所处**环境**，并**自主**地通过**执行器**采取**行动**以达成特定目标的实体。

# HelloAgents框架学习

## HelloAgentsLLM
目标：<br/>
1、多提供商支持：支持主流LLM，例如：OpenAI、ModelScope、智普AI<br/>
2、本地模型集成：VLLM、Ollama<br/>
3、自动检测机制：自动识别推断使用LLM<br/>
### 多提供商支持
让 HelloAgentsLLM 在内部处理不同服务商的配置细节，从而为用户提供一个统一、简洁的调用体验。
### 本地模型集成
本地模型存在的意义：本地实现高性能、生产级的模型推理服务<br/>
这些模型兼容 OpenAI 标准
### 自动检测机制
最高优先级：检查特定服务商的环境变量（.env中的配置）<br/>
次高优先级：根据 base_url 进行判断(域名匹配、端口匹配)<br/>
辅助判断：分析 API 密钥的格式，某些服务商的 API 密钥有固定的前缀或独特的编码格式。不过，由于这种方式可能存在模糊性（例如多个服务商的密钥格式相似），因此它的优先级较低，仅作为辅助手段。

## 框架接口实现
### Message类
定义框架内统一的消息格式，确保智能体与模型之间信息传递的标准化。
